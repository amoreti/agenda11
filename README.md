Agenda 11 do curso de Desenvolvimento de Sistemas.

Essa atividade consiste em um sistma simples de monitoramento de um reservatório de água.
O objetivo é exibir mensagens coloridas que representem diferentes situações de alerta para facilitar a visualização.
Para tal atividade, foi utilizada a biblioteca COLORAMA do Python.
